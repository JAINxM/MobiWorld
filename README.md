# MobiWorld - Advanced E-commerce Site

MobiWorld is a fully functional advanced level E-commerce website built with PHP (Session/Array based), Tailwind CSS, and JavaScript.

## 🚀 Features
- **Modern UI**: Clean, premium design with Tailwind CSS, Glassmorphism, and smooth animations.
- **Auth System**: Registration & Login using PHP Session storage.
- **Dynamic Catalog**: 10+ smartphones loaded dynamically from PHP arrays.
- **Shopping Cart**: Add to cart, Buy Now, dynamic total calculations with JavaScript.
- **Profile Dashboard**: User information, total spending, total orders, and full purchase history.
- **Admin Panel**: Statistics for revenue, phones sold, user registration, and transaction history with JS Charts.

## 📁 Technical details
- **Core**: PHP
- **Storage**: No Database. Uses `$_SESSION` and hardcoded arrays.
- **Design**: Tailwind CSS (CDN) + Font Awesome + Google Fonts (Outfit).
- **Charts**: Chart.js for admin analytics.

## 🛠️ Installation
1. Move the `MobiWorld` folder to your local server directory (e.g., `xampp/htdocs/`).
2. Open your browser and go to `http://localhost/MobiWorld/`.
3. Start by Registering an account.
